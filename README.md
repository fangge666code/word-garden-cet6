# 词间 · Word Garden

一个仅供个人使用的 CET-6 单词卡片网站。它会优先安排到期复习，再补充每日新词，并把学习进度保存在当前浏览器中。

## 功能

- 单词卡片翻面与“忘记／模糊／认识”评分
- 分级复习间隔与当日忘词重排
- 今日统计、连续学习天数与掌握进度
- CET-6 核心词库搜索与状态筛选
- 每日目标、减少动画、JSON 导入导出和清空记录
- 响应式布局与键盘操作

## 本地运行

项目不依赖第三方运行库。使用 Node.js 20 或更高版本：

```text
npm run dev
```

浏览器访问 `http://127.0.0.1:4173`。构建使用 `npm run build`，测试使用 `npm test`。

词库来源与许可说明见 `docs/WORDLIST.md`。
