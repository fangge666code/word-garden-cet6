# 词间 · Word Garden

一个支持大学英语六级与考研英语的单词卡片网站和 Android App。它会优先安排新词，再补充到期复习，并通过账号在设备间同步两套互相隔离的学习进度。

## 功能

- 单词卡片翻面与“忘记／模糊／认识”评分
- 分级复习间隔与当日忘词重排
- 今日统计、连续学习天数与掌握进度
- 3000 词六级模块与 3500 词考研模块，独立乱序、进度和每日目标
- 英式女声按需分块下载，未变化资源跨版本复用
- 词库搜索与状态筛选
- 账号登录与跨设备云同步
- 每日目标、减少动画、JSON 导入导出和清空记录
- 响应式布局与键盘操作

## 本地运行

使用 Node.js 22 或更高版本：

```text
npm run dev
```

浏览器访问 `http://127.0.0.1:4173`。构建使用 `npm run build`，测试使用 `npm test`。

词库来源与许可说明见 `docs/WORDLIST.md` 和 `THIRD_PARTY_NOTICES.md`。
