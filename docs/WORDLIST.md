# 词库说明

首版收录 60 个 CET-6 核心学习词条，用于验证完整的卡片学习、复习调度、搜索筛选和数据备份流程。单词范围参考 JavaProgrammerLB/cet-word-list 项目；该项目依据《全国大学英语四、六级考试大纲（2016 年修订版）》整理，并以 MIT License 发布。

词性、中文释义和例句为本项目整理的学习内容，不是对参考仓库文本的复制。后续可在保持相同数据结构的前提下扩充完整词库。
