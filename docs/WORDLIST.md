# 词库说明

当前版本收录 1500 个 CET-6 核心学习词条。原有 60 个核心词保持稳定 ID，其余词条从 MIT License 的 ECDICT 中筛选：只保留标记为 `cet6` 而未标记为 `cet4` 的单词，并按照 BNC 和当代语料库词频优先选择。

新增词条的音标和中文释义来自 ECDICT；例句由本项目模板生成。学习队列按日期进行确定性乱序，不复制新东方或其他商业词书的释义、例句、词根联想和编排顺序。
