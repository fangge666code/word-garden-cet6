# 词间分块离线发音设计

## 目标

网页与 Android App 在没有系统英语 TTS、没有浏览器语音输出的华为设备上，也能播放全部 3000 个 CET-6 单词。学习记录、账号同步和更新流程保持不变。

## 选择的方案

构建时使用开源 eSpeak NG 1.51 生成英式英语发音，去除多余静音并转换为 11025 Hz、8 位、单声道 PCM。每 100 个单词合并为一个 WAV 音频包，共 30 个包，并生成以单词稳定 ID 为键的采样位置索引。

相比 3000 个独立文件，分块方案减少了部署文件数量；相比单个大音频文件，网页首次点击只需下载约一个小包。Android App 会把全部音频包内置在安装包中。

## 播放流程

1. 用户点击发音按钮，应用根据单词 ID 查找音频包、起始采样和时长。
2. 使用 Web Audio API 获取并解码对应音频包；同一音频包只解码一次并在当前页面缓存。
3. 停止正在播放的上一个单词，从指定位置播放当前单词。
4. 内置音频加载或解码失败时，回退到 v1.2.0 的 Web Speech 方式；Android 原生 TTS 作为最后备用。
5. 所有方式都失败时显示明确提示，不再静默无声。

## 网页缓存

Service Worker 对已经播放过的音频包采用缓存优先策略。网页版第一次播放某个音频包需要联网，之后可从本地缓存播放；Android 安装版始终使用内置资源。

## 验证

- 校验 30 个 WAV 文件的 RIFF/WAVE 头、采样率、位深和单声道格式。
- 校验 3000 个索引全部落在对应音频包范围内。
- 单元测试覆盖内置音频播放、重复点击停止上一段、下载失败回退到旧版语音、完全不支持时的提示。
- 运行完整网页测试、构建和 Capacitor Android 同步。
- 发布 1.2.4，并在线验证版本清单、索引文件和随机音频包均可访问。

## 成功标准

- 华为手机不安装英语语音包也能听到 `wooden` 等单词。
- 网页只按需下载当前分块，不一次下载全部音频。
- Android APK 覆盖安装后保留原有学习记录。
