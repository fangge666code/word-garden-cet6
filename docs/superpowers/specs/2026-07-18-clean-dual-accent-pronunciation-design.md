# 纯净英美双口音发音设计

## 目标

为六级与考研共 6500 个单词提供无相邻词串音的英式、美式女声发音，并保持 Android 安装包轻量、音频按需下载和学习数据不变。

## 音频生成

- 英式使用 Kokoro `bf_emma`，美式使用微软神经女声 `en-US-AriaNeural`；两者都生成静态资源，用户播放时不依赖第三方语音服务。
- 每个单词形成独立缓存片段；允许为提高速度批量合成独立句子，但切片从单词自身时间戳开始，禁止保留上一词的前置声音。
- 现有英式缓存精确移除旧流程多出的 45 毫秒前置切片；新生成资源从单词自身时间戳裁剪，并应用淡入淡出，避免爆音和起音丢失。
- 缓存路径包含词库、口音、声音和生成版本；未变化且通过检查的单词可以复用。
- 生成任务允许并行执行，但写包和索引保持确定性。

## 资源组织与更新

- 六级英式、六级美式、考研英式、考研美式使用四套独立索引和资源目录。
- 每 25 个单词组成一个小型音频包，播放时只下载包含目标单词的资源包。
- 每个包使用内容哈希版本号；客户端保留未变化缓存并只替换发生变化的包。
- Android APK 不内置大体积音频资源。

## 用户界面

- 学习卡片和词库行均显示紧凑的“英”和“美”两个发音按钮。
- 两个按钮具有独立的无障碍标签，并阻止点击发音时触发卡片翻面或其他操作。
- 当前播放按钮显示状态；切换口音时立即停止上一段音频。
- 学习卡片出现时预加载当前词和下一词的英美小包，手机点击时直接播放。
- 登录后的今日页不再展示账号同步、检查更新和六级/考研选择，只保留今日任务与进度。
- 账号同步、检查更新和六级/考研选择集中到设置页；登录前首页仍保留登录注册入口。

## 回退与错误处理

- 英式资源失败时回退到设备 `en-GB` 语音，美式资源失败时回退到设备 `en-US` 语音。
- 浏览器语音不可用时继续使用 Android 原生语音能力。
- 单个资源失败不影响学习流程，并显示可理解的失败提示。

## 质量检查

- 每个词条必须有英式和美式索引，且映射到有效 PCM 片段。
- 检查文件非空、采样率、声道、异常时长、削波、首尾爆音和静音。
- 重点抽检 `ocean` 以及元音开头、爆破音开头、辅音连缀和多音节单词。
- 运行现有测试并新增双口音选择、回退和资源完整性测试。

## 发布

- 作为新的补丁版本发布网页与 Android APK。
- 学习记录、账号、六级/考研模块和云端数据表均不迁移、不清空。
